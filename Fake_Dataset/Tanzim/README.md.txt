# Fake Dataset

This folder is reserved for synthetic or manipulated license plate images.

Currently, no fake images have been added.

The fake dataset will contain:
- AI-generated license plates
- Augmented images
- Inpainted or modified samples
- Adversarial or tampered examples

These images will be used for:
- Model robustness testing
- Forensic verification experiments
- Human-in-the-loop validation
